## 一、价卡基础（仅适用日本跨境直邮店铺）

> ⚠️ **范围声明**：本价卡和本报告所有计算**仅适用于"跨境直邮店铺"**，不包括日本本土店和全托管店铺。MVP 阶段只做跨境直邮，未来扩展到本土店/全托管店时，需要新增对应的价卡 JSON 文件和店铺类型字段。

**计费规则**

- 50g 起重，按 g 计费，超 30kg 按 20–30kg 阶梯结算
- 按**实重**计费（不计体积重）
- 计费节点：妥投 / 派送异常 Fail Delivery
- 单包裹 ≤ 30kg，最大尺寸 ≤ 60×50×40 cm
- **运费公式**：`freight_jpy = per_parcel + per_kg × 实重(kg)`

## 二、ROI 计算公共参数

| 参数 | 值 | 说明 |
|---|---|---|
| 平台佣金率 | 5% | TikTok Shop 日本站直邮，类目可能有差异，按基础值估算 |
| 退货损耗率 | 4% | 行业经验值 |
| 包装/操作费 | 30 JPY/单 | 贴单 + 内袋 |
| 汇率 | 1 RMB ≈ 21 JPY | 当前参考值，正式版应接实时汇率 API |
| ROI 保本线 | 2.5 | 本项目核心决策指标 |

**ROI 公式**

```
变动成本 = 货源价×汇率 + 头程×汇率 + 运费 + 包装
实收     = 售价 × (1 - 佣金率) × (1 - 退货损耗率)
毛利     = 实收 - 变动成本
ROI 保本时广告预算 ≤ 毛利 / 2.5
```

## 七、本次测试发现的问题与改进项

### 7.1 FastMoss 接入

- ✅ Base URL：`https://openapi.fastmoss.com`
- ✅ 认证：`Authorization: Bearer <api_key>`
- ✅ Method：**POST + JSON body**（不是 GET query！文档明确）
- ✅ 商品搜索 `/product/v1/search` 调通，能拿到真实日本商品的售价/销量/评分/类目/店铺
- ⚠️ **pagesize 上限是 10**（>10 返回空数组，文档默认 10 也佐证）
- ⚠️ **关键词只支持单 token**，多个空格分隔的关键词返回 `total=0`
- ⚠️ **试用 key quota 极小**：本次跑 ~30 次调用就触发 `code=3 quota limit`

## 八、附：FastMoss 调用样例

```bash
curl -X POST 'https://openapi.fastmoss.com/product/v1/search' \
  -H 'Authorization: Bearer <api_key>' \
  -H 'Content-Type: application/json' \
  -d '{
    "keywords": "化粧ポーチ",
    "filter": {"region": "JP"},
    "orderby": [{"field": "day7_units_sold", "order": "desc"}],
    "page": 1,
    "pagesize": 10
  }'
```

**响应字段**（精简）

```
data.list[].product_id          - 商品ID
            .title              - 标题
            .region             - 区域 (JP/US/...)
            .currency           - 币种 (JPY/USD/...)
            .price              - 售价（字符串，可能含区间）
            .total_units_sold   - 总销量
            .total_gmv          - 总 GMV
            .day7_units_sold    - 近7天销量
            .day28_units_sold   - 近28天销量
            .day90_units_sold   - 近90天销量
            .product_rating     - 评分
            .commission_rate    - 佣金率（如 "10%"）
            .creator_count      - 关联达人数
            .video_count        - 关联视频数
            .category           - {l1, l2, l3}
            .shop               - {seller_id, name, avatar, total_units_sold}
            .is_cross_border    - 是否跨境
            .ctime              - 上架时间
```
