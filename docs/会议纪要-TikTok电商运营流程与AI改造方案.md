# TikTok 跨境电商运营会议纪要

> 会议参与者：张栩尧（运营）、李总、及其他团队成员
> 店铺类型：日本跨境直邮店（白牌/普货）

---

## 一、当前工作流程梳理

### 1. 选品与上品

**标题生成流程：**
- 在 TikTok 上找 10 条同款产品链接
- 提取高频关键词（如：自动折叠、12骨、晴雨兼用、男女兼用等）
- 结合个人试用的主观感受，补充差异化卖点（如：可挂式设计、便携等）
- 用 AI（豆包等）将关键词整合为一条完整的产品标题
- 平台自动翻译为日文

**类目与属性填写：**
- 手动选择商品类目（如美妆个护等细分类目）
- 品牌选择「无品牌」（白牌店铺）
- 材质、参数等需要去 1688 搜同款，逐一对照填写

**商品描述：**
- 方式一：将产品介绍给 AI，生成商品描述
- 方式二：从 10 条竞品链接中选最好的详情页，下载图片后上传到店铺

**SKU 与变体设置：**
- 手动设置颜色、尺码等销售变体
- SKU 编码规则：产品名-颜色-数量（如：卷发棒-白-1）
- 不同 SKU 组合（单个/5条装/10条装）影响运费和利润，需逐一计算

### 2. 定价与运费计算

**成本结构（四大成本项）：**

| 成本项 | 来源 | 说明 |
|--------|------|------|
| 货源成本 | 1688 或创世跨境对接的厂家 | 固定成本 |
| 物流运费 | TikTok 跨境直邮商家运费表 | 按克重分段计费，含中国到日本全程 |
| 投流成本 | TikTok 店铺广告（GM Max 等） | 变量，可调节 |
| 达人佣金 | 联盟合作 | 按最终交易金额的 10%~30% |

**运费计算方式：**
- 公式：产品克重 × 每公斤单价 + 基础包裹运费（475日元）
- 示例：0.2kg × 195日元 + 475日元 = 514日元
- 包邮 vs 不包邮：不包邮时买家承担 350 日元，商家承担差额；包邮时商家承担全部
- 运费按普货/特货/敏货有不同费率

**定价策略：**
- 先设较高零售价，再通过活动折扣制造"降价感"（先涨后折）
- 不同 SKU 可设不同价格（畅销色高价 or 低价引流）
- 核心指标：ROI 不低于 2.5（保本线）

### 3. 营销与促销活动

**自主促销工具：**
- 秒杀活动（3天限时，饥饿营销，转化最高）
- 商品折扣（日常，为期一个月）
- 优惠券（如 300 日元评价券，提升好评率）
- 包邮活动

**活动续期痛点：**
- 活动有开始/结束时间，到期需手动"续期"
- 比如 4月1日~4月4日 00:00 结束后，需手动创建 4月4日 00:01~4月7日
- 平台无到期提醒功能

**官方营销活动：**
- 平台提供 6 个板块的活动（拉达人、投流等）
- 新手运营看不懂活动规则，容易盲投

### 4. 投流与广告

**投流操作流程：**
- 创建广告计划 → 选择推广商品 → 设定目标 ROI → 设定每日预算
- 可投放自拍视频、达人视频、商品卡
- 手动模式下挑选优质视频集中预算（vs 自动模式平均分配）

**验证周期：**
- 每个产品的最佳 ROI 值需要约 3 天投流测试才能确定
- 每天需监控预算消耗速度、出单量，调整第二天的预算/ROI/售价

**关键公式：**
- 投流 ROI = 总收入(GMV) / 投流成本
- 平均下单成本 = 总投流花费 / 总出单数
- 示例：花费 $1000，出 522 单，单转成本 ≈ $2

### 5. 达人合作与联盟

**合作模式：**
- 公开合作：商品放入联盟池，所有达人可自由带货
- 定向合作：主动寻找特定达人，设置佣金、样品政策

**达人筛选痛点（最耗时环节之一）：**
- 需逐一打开达人主页查看：带货转化率、视频播放量、合作历史
- 平台数据非实时，需去 TikTok APP 查看近期真实播放数据
- 达人数据方差大（可能两条视频 2000 万播放，其他只有几百）
- 平台有邀请上限（约 200 个/批次）

**视频授权流程：**
- 发现达人爆款视频后，需私信达人获取"视频代码"
- 只有拿到授权码才能对该视频投流
- 部分达人不回复消息，需用话术引导（高佣金、免费样品、涨粉等）

### 6. 订单管理与打包

- 订单按时间排列，不同产品混杂在一起
- 无法按 SKU 筛选/分组
- 面单打印后需人工分拣，按产品类别分配给不同打包人员
- 订单量大时（如日出 100 单），分拣效率极低

## 二、核心痛点汇总

### P0 级痛点（核心价值，优先解决）

| # | 痛点 | 当前做法 | 影响 |
|---|------|----------|------|
| 1 | **ROI 计算与定价决策** | 手动用 Excel 算运费、成本、利润，凭经验定价 | 定价不科学，新手易亏损 |
| 2 | **竞品价格分析** | 无法系统获取同品类竞品定价和销量数据 | 定价缺乏市场依据，可能定高无人买或定低亏钱 |
| 3 | **达人筛选与评估** | 逐个打开达人主页，人工看数据、对比、做判断 | 5分钟/人，效率极低，且判断偏主观 |
| 4 | **投流 ROI 测试** | 需 3 天手动观察调整，逐日看消耗曲线 | 试错成本高，新品上线周期长 |

### P1 级痛点（效率提升，锦上添花）

| # | 痛点 | 当前做法 | 影响 |
|---|------|----------|------|
| 5 | **商品标题生成** | 手动搜 10 条链接提取关键词，再拼接 | 重复劳动，可自动化 |
| 6 | **商品属性填写** | 去 1688 搜同款，逐项对照填写材质/参数 | 机械操作，耗时 |
| 7 | **活动续期提醒** | 手动记录活动到期时间，手动续期 | 容易遗忘，活动中断影响销量 |
| 8 | **订单分拣** | 打印面单后人工按产品分类 | 订单多时效率低、易出错 |
| 9 | **官方活动选择** | 凭经验判断是否参加平台活动 | 新手运营看不懂，容易盲投 |
| 10 | **视频素材生成** | 自拍或依赖达人产出 | 内容产能瓶颈 |

## 三、AI 整体改造方案

### 整体架构：以 ROI 为核心的 AI 运营助手

```
┌─────────────────────────────────────────────────┐
│              TikTok ROI 智能运营平台               │
├────────┬────────┬────────┬────────┬──────────────┤
│ 智能定价 │ 达人评估 │ 投流优化 │ 内容生成 │  运营自动化 │
│  引擎   │  系统   │  助手   │  工具   │    工具集     │
├────────┴────────┴────────┴────────┴──────────────┤
│                   数据采集层                       │
│     (Buzzmore/竞品数据/1688/TikTok后台数据)        │
└─────────────────────────────────────────────────┘
```

### 模块一：智能定价引擎（P0）

**解决痛点：** ROI 计算、竞品分析、定价决策

**功能设计：**
- 输入货源成本、产品克重，自动计算物流成本
- 结合达人佣金比例、投流预算，反推最优售价
- 设定 ROI 保本线（如 2.5），自动校验定价是否合理
- 对接竞品数据（Buzzmore 等），给出市场价格区间和建议
- 输出：建议零售价、折扣价、秒杀价、各 SKU 差异化定价方案

**实现路径：**
1. 先做本地计算工具（运费计算器 + ROI 模拟器）
2. 接入 Buzzmore API 或爬取竞品数据
3. 开发浏览器插件，在 TikTok 后台自动填写价格

### 模块二：AI 达人评估系统（P0）

**解决痛点：** 达人筛选耗时、判断主观

**功能设计：**
- 批量抓取达人数据：粉丝数、带货GMV、视频播放量、合作品牌数
- 计算达人稳定性指标（播放量方差/标准差，过滤"偶发爆款型"达人）
- 根据产品类目匹配最佳达人，输出推荐排名
- 自动生成达人邀约话术（针对不同达人特点定制）
- 标记已邀请/已合作/待跟进的达人状态

**实现路径：**
1. 从 TikTok 后台联盟数据导出达人列表
2. 补充 Buzzmore 达人维度数据
3. 建立达人评分模型（转化率 × 稳定性 × 品类匹配度）

### 模块三：投流优化助手（P0）

**解决痛点：** 投流测试周期长、调参靠经验

**功能设计：**
- 根据产品成本结构，预测初始最优 ROI 目标值
- 实时监控投流数据（消耗速度、出单曲线），自动预警
- 基于历史投流数据，给出预算分配建议（哪些视频值得加投）
- 模拟不同 ROI 目标下的盈亏平衡点

**实现路径：**
1. 每日导出投流数据，建立产品-投流效果数据库
2. 训练简单预测模型（类目 × 价格段 × 达人类型 → 预测转化率）
3. 开发投流仪表盘，可视化关键指标

### 模块四：内容生成工具（P1）

**解决痛点：** 视频素材产能不足、标题生成低效

**功能设计：**
- 智能标题生成：输入产品链接，自动提取关键词 + 生成多版本标题
- 爆款视频脚本分析：分析头 3 秒 hook、转化结构，生成新脚本
- AI 视频生成：对接即梦/Seedance 等模型，批量生成测试视频
- 商品描述自动生成：基于 1688 详情 + 卖点，生成多语言描述

### 模块五：运营自动化工具集（P1）

**解决痛点：** 重复操作、活动管理、订单分拣

**功能设计：**
- 商品属性自动填充：从 1688 链接提取参数，自动映射到 TikTok 字段
- 活动到期提醒 + 一键续期
- 订单智能分拣：按 SKU 自动分组，生成分拣清单给打包人员
- 官方活动推荐：分析当前商品特点，推荐最适合的平台活动并预估效果
- 浏览器插件：在 TikTok 后台自动填写价格、SKU、属性等字段

## 五、关键数据源

| 数据源         | 用途           | 获取难度          |
| ----------- | ------------ | ------------- |
| TikTok 卖家后台 | 订单、投流、达人合作数据 | 中（需手动导出或插件抓取） |
| Buzzmore    | 竞品价格、销量、达人数据 | 低（付费会员可获取）    |
| 1688        | 货源成本、商品参数    | 低（公开数据）       |
| TikTok 运费表  | 物流成本计算       | 低（官方提供，定期更新）  |
| TikTok APP  | 达人真实近期数据     | 高（无网页端，反爬严格）  |

## 六、会议核心结论

1. **整套运营逻辑始终围绕 ROI 运转**，所有动作（定价、投流、达人合作、活动参与）最终目标都是实现既定 ROI
2. **AI 最大价值不在操作自动化，而在决策支持**——帮运营人员算清楚"该怎么定价、该选哪个达人、该投多少流"
3. **操作自动化是锦上添花**（浏览器插件自动填写等），核心价值是数据驱动的决策引擎
4. **优先研究 Buzzmore 数据源**，解决竞品数据获取问题（比自己爬 TikTok 现实得多）
5. **Phase 1 立即启动**：先做 ROI 计算工具，这是团队当前最痛的点
